# Markerless Multicam Organizer

다중 카메라 동기화 촬영 영상을 자동 매칭, 프리뷰, 리네임, 검증, 정리하는 GUI 도구입니다.

## 배경

스포츠 현장에서 **마커리스(markerless) 동작분석**을 수행할 때, 다수의 카메라로 동시에 프레임 동기화 촬영을 진행합니다. 촬영 후 각 카메라의 SD 카드를 PC로 복사하면, 카메라마다 파일명 규칙과 내부 시계가 달라 어떤 영상이 같은 시점의 촬영인지 파악하기 어렵습니다.

이 도구는 **영상 길이(duration) 기반 자동 매칭**으로 이 문제를 해결합니다.

### 분석 워크플로우

```
촬영 (ZCAM E2 4K x N대)
  → 이 도구로 영상 정리 (매칭 · 리네임 · 검증)
    → Theia3D (마커리스 3D 자세 추정)
      → Visual3D (운동학·운동역학 분석)
```

### 사용 장비

- **카메라**: ZCAM E2 4K Original (다중 대수, 프레임 동기화 촬영)
- **분야**: 운동역학(Biomechanics), 스포츠 과학

## 주요 기능 (6단계 워크플로우)

| 단계 | 기능 | 설명 |
|------|------|------|
| 1 | 스캔 & 매칭 | 카메라별 폴더 스캔, duration 기반 세트 자동 매칭 |
| 2 | 세트 프리뷰 | 매칭된 세트를 그리드 뷰어로 프레임 단위 비교 확인 |
| 3 | 리네임 | 통일된 파일명(C0001.mp4, C0002.mp4 ...)으로 일괄 변경 |
| 4 | 분석 | 리네임된 파일의 메타데이터 그룹별 수집 |
| 5 | 검증 | 세트별 duration, 프레임 수, 해상도, FPS 동기화 검증 |
| 6 | 정리 | 세트 단위 폴더 구조로 복사 또는 이동 |

### 세트 프리뷰어 기능

- 그리드 레이아웃으로 모든 카메라 프레임 동시 표시
- 프레임 스텝 선택 (1, 5, 10, 15, 30, 50 프레임)
- 재생 속도 조절 (0.25x, 0.5x, 1x, 2x)
- 세트 목록에서 개별/다중 선택 프리뷰

## 요구 사항

- Python 3.8+
- OpenCV: `pip install opencv-python`
- Pillow: `pip install Pillow`

```bash
pip install opencv-python Pillow
```

## 사용법

```bash
python organization_mp4_v2.py
```

1. **폴더 선택**: 카메라별 하위 폴더(cam01, cam02, ...)가 있는 루트 폴더 선택
2. **1~6 단계 버튼**을 순서대로 클릭하여 진행

### 폴더 구조 예시

```
root/
├── cam01/          # 카메라 1 SD카드 복사본
│   ├── MVI_3421.mp4
│   ├── MVI_3422.mp4
│   └── ...
├── cam02/          # 카메라 2 SD카드 복사본
│   ├── CLIP0087.mp4
│   └── ...
└── cam03/
    └── ...
```

정리 완료 후:

```
root/OrganizedVideos/
├── C0001/
│   ├── cam01/C0001.mp4
│   ├── cam02/C0001.mp4
│   └── cam03/C0001.mp4
├── C0002/
│   ├── cam01/C0002.mp4
│   └── ...
└── ...
```
